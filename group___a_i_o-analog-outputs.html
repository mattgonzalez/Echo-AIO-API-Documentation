Sets the gain value for the specified output channel. Output gain control is deprecated but still supported for backwards compatibility. Gain values range from 0 to 255. To match the 10x setting on the console, set the gain value to 255 For the 1x console setting, set the gain value to 26.

Parameters
outputChannelOutput channel number, starting at 0
gainGain value
Returns
AIO_OK if successful
