Configure AIO-C module

Description

AIO-C configuration constants and parameters.

To configure an AIO-C module, call AIO_setModuleIntParameter and AIO_setModuleDoubleParameter to set one of the parameters listed below. Note that some parameters require an integer value and that others require a double-precision floating point value. You can also use AIO_getModuleIntParameter or AIO_getModuleDoubleParameter to read the current value of a parameter.

void configureAIOCModule()
{
//
// Read the AIO-C module firmware version
//
int firmwareVersion = 0;
int status = AIO_getModuleIntParameter(AIO_comboParameterFirmwareVersion, &firmwareVersion);
if (status != AIO_OK)
{
return;
}
//
// Enable the 5 VDC power supply
//
status = AIO_setModuleIntParameter(AIO_comboParameter5VDCEnable, 1);
if (status != AIO_OK)
{
return;
}
//
// Set the module battery simulator to measure up to 250 mA
//
status = AIO_setModuleIntParameter(AIO_comboParameterMeasuredCurrentRange, AIO_comboCurrentMeasurementRange_250mA);
if (status != AIO_OK)
{
return;
}
std::cout << "AIO-C firmware version: " << firmwareVersion << std::endl;
}

Python example for AIO-C module configuration
Below is an example shows how to configure an AIO-C module using the EchoAIOInterface.dll.
API calls in this example make use of the Python API interface. The Python API interface is a wrapper around the C++ API interface. The Python API interface is provided as a convenience for Python programmers. It exposes the enums and defines from the C++ API interface

First, download the Python API interface files at https://github.com/mattgonzalez/EchoAIOExample
To use the inteface with the AIO Combo Module API, copy the file EchoAIOInterface_ComboModule_Python.py to the same directory as the python file, and import the file into your script.

#
# ctypes provides C compatible data types, and allows calling functions in DLLs or shared libraries.
#
import ctypes
import EchoAIOInterface_ComboModule_Python as combo
#
# Change the following path to the location of EchoAIOInterface.dll.
# The default location is C:\Progam Files\Echo AIO\CLI + API\EchoAIOInterface.dll
#
path = 'C:\Program Files\Echo AIO\CLI + API\EchoAIOInterface.dll'
echo = ctypes.WinDLL(path)
#
# Include the AIO_combulemodule Enumeration Classes and defines
#
import EchoAIOInterface_ComboModule_Python.py as AIOpy
#
# Define the ctypes function arguments and return types
#
echo.AIO_getModuleIntParameter.argtypes = [ctypes.c_int ,ctypes.c_int, ctypes.POINTER(ctypes.c_int)]
echo.AIO_getModuleIntParameter.restype = ctypes.c_int
echo.AIO_setModuleIntParameter.argtypes = [ctypes.c_int, ctypes.c_int, ctypes.c_int]
echo.AIO_getModuleDoubleParameter.argtypes = [ctypes.c_int, ctypes.c_int, ctypes.POINTER(ctypes.c_double)]
echo.AIO_getModuleDoubleParameter.restype = ctypes.c_int
#
# Initialize the EchoAIOInterface library
#
echo.AIO_initialize()
#
# Read the AIO-C module firmware version
#
firmwareVersion = ctypes.c_int()
print(echo.AIO_getModuleIntParameter(1, combo.AIOComboModuleIntParameters.AIO_comboParameterFirmwareVersion, ctypes.byref(firmwareVersion)))
print(hex(firmwareVersion.value))
#
# Set the AUX OUT pins to 0x03
#
echo.AIO_setModuleIntParameter(1, combo.AIOComboModuleIntParameters.AIO_comboParameterAuxout, 3)
#
# Print the value of the AUX OUT pins
#
auxOut = ctypes.c_int()
echo.AIO_getModuleIntParameter(1, combo.AIOComboModuleIntParameters.AIO_comboParameterAuxout, ctypes.byref(auxOut))
print(hex(auxOut.value))
#
# Set the module battery simulator to measure up to 250 mA
#
echo.AIO_setModuleIntParameter(1, combo.AIOComboModuleIntParameters.AIO_comboParameterMeasuredCurrentRange, combo.AIOComboCurrentMeasurementRanges.AIO_comboCurrentMeasurementRange_250mA)
#
# Shutdown the EchoAIOInterface library
#
echo.AIO_shutdown()
